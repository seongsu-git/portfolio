# 🚀 최성수 | 프론트엔드 개발자 포트폴리오

---

## ✨ Key Features

* 📱 Responsive Design: 모바일, 태블릿, 데스크탑 전 기기 대응
* 🌙 Dark Mode: 시스템 설정 및 사용자 선택에 따른 다크모드 지원
* ⚡ Optimization: SEO 최적화

---

## 🛠 Tech Stack

* Framework: Next.js 15 (App Router), React
* Language: TypeScript
* Styling: CSS Modules
* Deployment: Vercel

---

## 📂 Project Structure

.
├── app/             # Next.js App Router
├── components/      # UI 컴포넌트
├── public/          # Images, Favicons
├── styles/          # CSS
└── constant/        # 정적데이터 및 타입 정의

## ⚙️ Development

# 의존성 설치
npm install

# 로컬 서버 실행
npm run dev
